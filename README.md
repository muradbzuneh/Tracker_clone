# 🏋️‍♂️ FitTrack – Fitness Tracker App

FitTrack is a modern fitness tracking web application built with React, TypeScript, and Tailwind CSS.
It allows users to track their daily activities, food intake, and personal fitness goals in a clean and responsive UI.

This project was built as part of my journey to improve my TypeScript skills while following and extending a full-stack fitness tracker tutorial.

## 📌 Features

* 🔐 Authentication System (Login / Signup)
* 👤 User Profile Management
* 📊 Dashboard with Stats
* 🍎 Food Logging
* 🏃 Activity Tracking
* 🎯 Goal-based Calorie Calculation
* 🌙 Dark Mode UI
* 📱 Responsive Design

## 🛠️ Tech Stack

* Frontend: React + TypeScript
* Styling: Tailwind CSS
* Routing: React Router
* State Management: Context API
* Icons: Lucide React
* Notifications: React Hot Toast
* Mock Backend: Custom mock API

## 📂 Project Structure

```
src/
│── assets/        # UI components & mock API
│── context/       # Global state (AppContext)
│── pages/         # Main pages (Login, Dashboard, Profile, etc.)
│── types/         # TypeScript types
│── components/    # Reusable UI components
```

## ⚙️ Installation & Setup

```bash
# Clone the repository
git clone (https://github.com/muradbzuneh/Tracker_clone.git)

# Navigate into project
cd frontend

# Install dependencies
npm install

# Run development server
npm run dev
```

---

## 📦 Build

bash
npm run build
